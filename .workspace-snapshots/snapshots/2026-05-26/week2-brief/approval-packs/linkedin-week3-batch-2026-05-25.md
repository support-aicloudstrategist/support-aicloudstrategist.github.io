## Day 16 — DPDP for diagnostic labs — the test report consent gap

Suggested posting time: 11:20 AM IST

Decision sought: APPROVE / REJECT / EDIT

Diagnostic labs handle some of the most sensitive everyday data: names, phone numbers, test details, reports, and sometimes family-member coordination.

The consent gap often appears in small places:
- report delivery on WhatsApp
- shared phone numbers
- online booking forms
- repeat reminders
- staff access to old reports

DPDP readiness is not about making work harder. It is about making data handling clear, documented, and respectful.

Soft CTA: If you run a lab, start by mapping how a report moves from booking to delivery.

Five mistakes I often see on clinic listings:

1. Outdated phone or timing information.
2. No clear WhatsApp path.
3. Weak doctor/clinic description.

Practo and Justdial can create discovery. But discovery is not conversion.
